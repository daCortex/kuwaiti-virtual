# Kuwaiti Virtual

An independent, fan-made Middle-Eastern virtual airline for flight simulation
(Infinite Flight), inspired by **Kuwait Airways** — professionalism, realism and
true Arabian hospitality. Built with Next.js 16, React 19 and Tailwind CSS v4.

> Kuwaiti Virtual is not affiliated with, endorsed by, or sponsored by the
> real-world airline Kuwait Airways, or by Infinite Flight.

## Features

A full virtual-airline platform centred on the **Crew Center** (`/crew`):

- **Crew Center** — staff dashboard with Applications, Pilots roster, PIREP review,
  live Flights, Stats/leaderboard, LOA management, Reports and News admin.
- **Pilot hub** — file PIREPs (with one-click autofill from your IF logbook),
  flight deck, rank progression and leaderboard.
- **Public site** — home, about, staff, ranks, operations/network, news, codeshares,
  fleet explorer and a live flight map.
- **Theming** — light (default) and a deep Classic-Dark-Blue dark mode,
  driven entirely by CSS design tokens.

recognition programme. Typeface: Inter.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

The app runs out-of-the-box in **demo mode** (CEO account only — the roster starts
empty). It goes live the moment you configure the optional integrations:

- `DATABASE_URL` — Neon Postgres for pilots + PIREPs (otherwise in-memory demo store).
- `IF_API_KEY` — Infinite Flight Live API for the live map and logbook autofill.
- Discord OAuth / bot env vars for pilot login and the Discord integration.

## Data

- `src/lib/data.ts` — single source of truth (brand, fleet, hubs, ranks, staff, …).
- `src/lib/routes.ts` — route network, generated by `scripts/gen-kuva-routes.mjs`.
- `src/lib/seed.ts` — roster seed (CEO only).
